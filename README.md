# 🌐 Pallavi Chougale | Portfolio Website

A responsive and modern **personal portfolio website** built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and contact details.

---

## ✨ Features

- Responsive single-page design
- Fixed navigation bar
- Smooth scrolling
- Hero / intro section
- About Me section
- Skills section
- Projects showcase
- Contact form
- Mobile-friendly layout
- Clean modern UI

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome**
